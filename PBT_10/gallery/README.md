# Infinite Scroll Gallery

## API đã dùng

- Lorem Picsum: `https://picsum.photos/v2/list`

## Cách chạy

1. Mở `index.html` bằng trình duyệt.
2. Trang sẽ tự động load 20 ảnh đầu tiên.
3. Cuộn xuống để tự động load thêm ảnh.
4. Click vào ảnh để mở lightbox hiển thị ảnh lớn.

## Tính năng

- Infinite scroll theo page.
- Lazy loading ảnh bằng `IntersectionObserver`.
- Lightbox modal khi click ảnh.
- Responsive layout: 4 cột desktop, 2 cột tablet, 1 cột mobile.
