# User Directory

## API đã dùng

- JSONPlaceholder: `https://jsonplaceholder.typicode.com`

## Cách chạy

1. Mở `index.html` bằng trình duyệt.
2. Chờ load danh sách user.
3. Dùng form để tạo, cập nhật, hoặc xóa user.
4. Nhập từ khoá vào ô tìm kiếm để lọc theo tên/email.

## Tính năng

- READ: tải danh sách user và hiển thị bảng.
- CREATE: thêm user mới bằng POST.
- UPDATE: sửa user hiện có bằng PUT.
- DELETE: xóa user bằng DELETE.
- SEARCH: lọc client-side theo tên/email.
- Loading state và thông báo lỗi/ thành công.
